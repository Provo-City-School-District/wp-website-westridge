## Westridge Elementary Website (Wordpress)
- [Westridge Elementary Website](https://westridge.provo.edu/)


You'll need to get the following directories and Database from Backup:
- /wp-content/uploads
- /wp-content/plugins
- .env file is in password manager

## .env Requirements
```
MARIADB_ROOT_PASSWORD=''
MYSQL_USER=''
MYSQL_PASSWORD=''
```

## Plugins Required
- Advanced Custom Fields
